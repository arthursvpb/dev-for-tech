# TypeScript

Install typescript

```sh
npm i -D typescript
```

Init typescript configuration file

```sh
./node_modules/typescript/bin/tsc --init
```

Run typescript compiler

```sh
./node_modules/typescript/bin/tsc
```

Install ts-node-dev -> Nodemon for TypeScript

```sh
 npm i -D ts-node-dev
```

Install node types

```sh
 npm i -D @types/node
```
